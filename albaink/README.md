# ALBA inK (알바잉크) — 유학생 아르바이트 매칭 플랫폼 프로토타입

> **inK = in Korea.** 한국에 들어와 있는 외국인 유학생을 대상으로 한 **합법·안전·다국어** 알바 매칭 플랫폼.
> 잉크방울 캐릭터 **ink(인코)** — “한국에서 유학생의 삶을 새롭게 기록한다.”

클릭 가능한 **HTML 프로토타입**입니다. 빌드 도구 없이 브라우저에서 바로 열어 화면 흐름과 핵심 기능을 확인할 수 있습니다.

## 실행

```
albaink/index.html   # 시작 페이지 (모바일앱 / 관리자 콘솔 진입)
```

`index.html`을 브라우저로 열면 됩니다. (별도 서버 불필요)

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | **홍보용 홈페이지(랜딩)** — 히어로·문제/솔루션·핵심기능·이용방법·대상별 가치·경쟁비교·브랜드 스토리·요금/제휴·CTA·푸터, KO/EN 토글 |
| `app.html` | **모바일앱** — 인트로 · 휴대폰 가입(SMS) · 구직자/구인자 마이페이지 · 출결 캘린더 |
| `admin.html` | **관리자 웹/앱** — 대시보드 · 매칭 현황 · 구인/구직 리스트 · 출결 현황판 · 준법 모니터링 |
| `start.html` | 간단 진입 런처(모바일앱/관리자 바로가기) |
| `styles.css` | 공유 디자인 시스템 (브랜드 컬러·컴포넌트) |
| `assets/ink.svg` | 캐릭터 ink(인코) |

## 홍보용 홈페이지 (index.html)

서비스 공개 랜딩 페이지. 전 구간 **KO/EN 언어 토글**이 동작합니다.

- **히어로** — 핵심 메시지 + 앱 미리보기(폰 목업) + CTA(앱 체험 / 제휴 문의)
- **시장 통계 스트립** — 253,434명 · CAGR 9.6% · 수도권 55.7% · 위반 차단 99%+
- **문제 정의** — 고규제 시장의 3중 마찰(준법 공백·언어 장벽·매칭 인프라 부재)
- **핵심 기능** — 준법 게이트 · 양측 검증 · 다국어 · 프랜차이즈 제휴 · 전자계약/출결 · 졸업 후 비자
- **이용 방법** — 가입 → 검증 → 준법 매칭 → 근무/출결 4단계
- **대상별 가치** — 유학생 / 사업자 2분할
- **경쟁 비교** — 알바몬·알바천국·당근알바 vs 알바잉크
- **브랜드 스토리** — ink(인코) · “inK = in Korea”
- **요금·제휴** — 구직자 무료 + 사업자 구독(Basic/Pro/SaaS/프랜차이즈)

## 모바일앱 (app.html)

앱 개발계획서의 화면 구성을 그대로 반영했습니다.

- **인트로/온보딩** — 중앙 로고, 상단 [구직][구인] 탭, 우상단 언어팩 버튼(KO/EN), 하단 약관·개인정보 링크
- **가입/로그인** — 휴대폰 번호 + SMS 인증 + 비밀번호 (간편 가입)
- **구직자 마이페이지** — 기본정보(이름·국적·생년월일·주소·연락처·여권번호·여권 스캔) / 추가정보(학교·전공·학년·TOPIK) / 구직정보(희망지역·업종·근무조건) / 구직 리스트(현재·기존)
  - **비자 허용 근로시간 게이트** — D-2 학기 중 20h 등 사용량 표시
- **구인자 마이페이지** — 기본정보(수요처명칭·사업자등록번호·사업자등록증·주소·연락처) / 수요정보(할 일·필요인원·근무조건·4대보험·급여·복지) / 구인정보(현재·기존 구인 리스트) / FAB 새 공고
- **출결 캘린더** — 월 뷰, 출근/퇴근 체크, 지각·결근·휴무 마킹

## 관리자 웹/앱 (admin.html)

- **대시보드** — 신규가입·매칭률·재매칭률·이탈률, 월별 매칭 추이, 국적/지역 분포
- **매칭 현황** — 현재/과거, 지역·업종 필터, 상태별 카운트
- **구인 리스트** — 공고 상태/기간/지원자 수, 준법 통과 여부
- **구직 리스트** — 학생 검증·비자·허용시간·TOPIK
- **출결 현황판** — 사용자별 주간 출결, 출근율, 누락 알림, 증빙 (권한 분리 열람)
- **준법 모니터링** — 비자·시간·업종 룰 엔진의 위반 공고 자동 차단 내역 + 기준표

## 핵심 차별화 (기획서 기반)

1. **준법 게이트** — 비자·한국어·허용시간·업종을 실시간 자동 판정, 위반 공고 사전 차단
2. **양측 검증** — 사업자등록증 · 여권/외국인등록증 기반
3. **다국어** — 한·영 기본, 중·베·몽 확장 (프로토타입은 KO/EN 토글 동작)
4. **프랜차이즈 제휴** — 제한업종을 피한 F&B·도소매 안정 공급처
5. **생애주기** — 유학 → 알바 → 졸업 → 비자 변경 지원
